# ⚖️ TanyaLaw - Konsultasi Hukum Jadi Mudah

Selamat datang di repositori **TanyaLaw**. Ini adalah proyek website yang dirancang untuk menjembatani masyarakat dengan akses bantuan hukum yang cepat, transparan, dan profesional.

## 🚀 Fitur Utama
* **Konsultasi Online:** Hubungi pengacara langsung melalui chat atau video call.
* **Direktori Advokat:** Cari tenaga hukum berdasarkan spesialisasi dan lokasi.
* **Artikel Hukum:** Edukasi hukum harian untuk masyarakat awam.
* **Pengecekan Dokumen:** Unggah draf kontrak atau surat untuk ditinjau.

## 🛠️ Teknologi yang Digunakan
* **Frontend:** HTML5, CSS3 (Tailwind CSS), JavaScript
* **Deployment:** GitHub Pages

## 📂 Struktur Folder
* `index.html` - Halaman utama
* `/assets` - Gambar, logo, dan file desain
* `/css` - File styling
* `/js` - Logika interaktif website

## 📝 Cara Berkontribusi
Jika Anda ingin memperbaiki bug atau menambah fitur:
1. Fork repositori ini.
2. Buat branch baru (`git checkout -b fitur-keren`).
3. Commit perubahan Anda (`git commit -m 'Menambah fitur X'`).
4. Push ke branch tersebut (`git push origin fitur-keren`).
5. Buat Pull Request.
